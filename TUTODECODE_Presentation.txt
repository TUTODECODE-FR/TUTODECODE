🛠️ FONCTIONNALITÉS RÉELLES ET COMPATIBILITÉ OS
--------------------------------------------------------------------------------

1. GHOST AI (Intelligence Artificielle Locale) - [OPÉRATIONNEL]
   - Technologie : Intégration native avec le serveur local Ollama.
   - Usage : Chat privé sans aucune fuite de données vers l'extérieur.
   - Statut : Réel. Nécessite Ollama installé sur la machine hôte.

2. NETKIT : DIAGNOSTIC RÉSEAU EN DIRECT - [OPÉRATIONNEL]
   - Info Système : Lecture en temps réel (Hostname, CPU, IPs locales). 
   - Port Checker : Test de connexion réel via Sockets TCP sur n'importe quel hôte.
   - DNS Lookup : Résolution réelle via les serveurs DNS du système.
   - Compatibilité : Fonctionne sur macOS, Windows et Linux.

3. BIBLIOTHÈQUE DE SCRIPTS - [OPÉRATIONNEL]
   - Scripts Windows (BATCH/PS) : Reset stack réseau, Check santé disque SMART.
   - Scripts Unix (BASH) : Chiffrement AES-256 via OpenSSL, génération de mots de passe.
   - Fonction : Visualisation et copie rapide pour exécution immédiate sur l'OS cible.

4. PARCOURS DE FORMATION - [CONTENU RÉEL]
   - Modules complets : De Linux/Bash à la Cybersécurité, en passant par Docker, 
     le Réseau Cisco, Python et SQL.
   - Progression : Système de suivi réel sauvegardé localement.

5. LABORATOIRE DE SIMULATION - [ÉDUCATIF]
   - Simulations visuelles : Ping ICMP, DNS Recursion, TCP Three-way Handshake.
   - Sécurité : Simulateur d'injection SQL interactif pour l'entraînement.
   - But : Visualiser ce qui est normalement invisible dans les câbles.

6. BOÎTE À OUTILS SAV
   - Calculateur IP / Masques : Logiciel de calcul intégré.
   - Guides VPN (Tailscale/ZeroTier) : Procédures réelles pas à pas.
   - GPO Master Guide : Référence des stratégies Windows Server.
