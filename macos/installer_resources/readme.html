Notes de Version - 1.0.3

Cette installation inclut le moteur de découverte locale et les modules de laboratoire suivants :

  • Network Control : Simulation réseau à haute fidélité.
  • Security Lab : Environnement de test de vulnérabilités.
  • Ghost AI : Assistant pédagogique local (compatible Ollama).

Note : L'installation nécessite environ 150 Mo d'espace libre, hors modules AI optionnels.
