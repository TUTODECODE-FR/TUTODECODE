# Assets Structure

This directory is organized for professional asset management:

- `logo.png`: The primary brand logo (1024x1024 recommended).
- `icons/`: Contains platform-specific launcher icons.
  - Run `flutter pub run flutter_launcher_icons` to regenerate from `logo.png`.
- `splash/`: Contains splash screen images.
  - Run `flutter pub run flutter_native_splash:create` to update.
- `fonts/`: Custom typography.
